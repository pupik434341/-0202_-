# -0202_-
Описание приложения
Приложение предназначено для расчета стоимости телефонных разговоров по двум тарифным планам с учетом сверхнормативного потребления минут и автоматического формирования квитанций об оплате.

Функциональность
Расчет стоимости по двум тарифам:
1. Тариф 1: до 200 минут - 0.7 руб/мин, сверх нормы - 1.6 руб/мин
2. Тариф 2: до 100 минут - 0.3 руб/мин, сверх нормы - 1.6 руб/мин

Интерфейс пользователя включает:
- Выбор тарифа
- Ввод количества минут
- Поля для ФИО абонента
- Поля для Адреса абонента
- Кнопки "Рассчитать" и "Печать"

Как использовать
1. Запустите приложение в 1С:Предприятие
2. Введите ФИО плательшика
3. Выедите Адрес плательщика
4. Выберите тариф (1 или 2)
5. Введите количество используемых минут
6. Нажмите "Рассчитать" для просмотра итого к оплате и минут которые превышают сверх норму
7. Нажмите "Печать" для создания чека

Тестовые сценарии
1.	ввод  больших  чисел в поля ввода (максимально большое по длине поля)
2.	ввод отрицательных чисел в поля ввода
3.	пустые поля  ввода 

Технические требования
- Платформа 1С:Предприятие 8.3.27
- Конфигурация с основной формой и модулями обработки

Особенности реализации
- Валидация вводимых данных
- Автоматический расчет сверхнормативных минут
- Форматирование даты в чеке
